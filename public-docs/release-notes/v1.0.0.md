# Project CBM v1.0.0 Release Notes

## Release Summary

Project CBM v1.0.0 is the first public image release for Raspberry Pi 3 through Raspberry Pi 5 and Raspberry Pi 500-class systems.

This release provides a Raspberry Pi OS Lite-based Commodore emulation appliance using VICE 3.10 and the Project CBM console menu system.

## Project Background

Project CBM v1.0.0 was created as a modern Raspberry Pi OS Lite-based Commodore emulation image inspired in part by the appliance-style experience of Carmelo Maiolino's Combian64.

Project CBM is independently assembled and is not a fork of Combian64. The v1.0.0 release focuses on providing a ready-to-image Project CBM environment for Raspberry Pi 3, Raspberry Pi 4, Raspberry Pi 5, and Raspberry Pi 500-class systems.

## Downloads

This release includes three downloadable assets:

- `pcbm-v1.0.0-rpi3-5.img.xz` - compressed Raspberry Pi image
- `pcbm-v1.0.0-docs.zip` - offline documentation bundle
- `SHA256SUMS` - checksums for verifying the downloaded files

## Release Artifacts

### `pcbm-v1.0.0-rpi3-5.img.xz`

The compressed Raspberry Pi image for Raspberry Pi 3, 4, 5, and Pi 500-class systems.

Flash this file to a microSD card using Raspberry Pi Imager, BalenaEtcher, or another imaging tool that supports `.img.xz` files.

Do **not** copy the `.img.xz` file to the SD card like a normal document. It must be written/flashed as a disk image.

### `pcbm-v1.0.0-docs.zip`

Offline documentation bundle for this release.

This bundle includes the README, end-user documentation, flashing instructions, troubleshooting notes, support information, license notes, and release notes.

### `SHA256SUMS`

Checksum file used to verify the downloaded release assets.

Use this file to confirm that the image and documentation bundle downloaded correctly before flashing the image.

## Supported Hardware

- Raspberry Pi 3
- Raspberry Pi 4
- Raspberry Pi 5
- Raspberry Pi 500

## First Boot Behavior

On first boot:

1. Raspberry Pi OS boots in console mode.
2. The root filesystem should expand to fill the SD card.
3. Project CBM launches on the primary console.
4. Users can start supported Commodore machines from the menu.

## Included Features

- Raspberry Pi OS Lite base
- VICE 3.10 console/SDL2 build
- Project CBM menu system
- Commodore machine launchers
- Content folders for games, demos, music, programs, and ROMs
- USB import helper
- Samba file sharing support
- Network information and test tools
- BBS/modem support hooks using TCPser
- Safe shutdown and reboot menu options

## Known Notes

- Use a quality SD card, especially with Raspberry Pi 5 and Pi 500 systems.
- Ethernet is the easiest first network setup.
- WiFi is intentionally left unconfigured so each user can configure their own region and network.
- Controller behavior may vary by USB gamepad model.
- The Content menu launches files using the current default machine.

## Verification

After downloading the release assets, verify them before flashing.

On Linux or Raspberry Pi OS:

```bash
sha256sum -c SHA256SUMS
```

On macOS:

```bash
shasum -a 256 -c SHA256SUMS
```

Expected result:

```text
pcbm-v1.0.0-rpi3-5.img.xz: OK
pcbm-v1.0.0-docs.zip: OK
```

If either file does not return `OK`, download it again before flashing.

## Recommended Flashing Tools

- Raspberry Pi Imager
- BalenaEtcher
- Linux command line using `xzcat` and `dd`

## Upgrade Notes

This is the first public release, so there is no previous Project CBM release to upgrade from.
