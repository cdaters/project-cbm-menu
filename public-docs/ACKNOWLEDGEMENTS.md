# Acknowledgements

Project CBM was inspired in part by Carmelo Maiolino's Combian64 project, which helped show how compelling a fast-booting, appliance-style Commodore emulation environment could be on Raspberry Pi hardware.

Project CBM is not a fork of Combian64 and is not affiliated with or endorsed by Combian64 or Carmelo Maiolino. It is an independently assembled Raspberry Pi OS Lite-based image using VICE and a custom Project CBM menu system.

One of the goals of Project CBM is to provide a modernized Commodore emulation appliance experience for newer Raspberry Pi systems, including Raspberry Pi 5 and Raspberry Pi 500-class hardware.
